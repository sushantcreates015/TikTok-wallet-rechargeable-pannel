Quick local test:
1) open terminal in this folder
2) run: npm install
3) run: npm start
4) open: http://localhost:3000

Render deploy:
1) unzip this file
2) upload ALL extracted files to the ROOT of your GitHub repo
3) in Render use:
   Build Command: npm install
   Start Command: node server.js
4) if Render asks for Root Directory, leave it blank
